  2. Livre para contribuição

O que Joomla NÃO É?

  1. Muleta de quem não sabe programar...
  2. Para facilitar trabalho de:
    • Sobrinhos
    • Programadores
    • Preguiços
  3. Joomla! é sistematizar o controle de conteúdo.

Blza, e Joomla o que é então?

  1. CMS - Sistema de gerenciamento de conteúdo
  2. Software de código aberto
    • Roda sob licença GNU versão 2
    • Desenvolvido em PHP com MySQL

Um pouco mais sobre o Joomla.

  1. Facilita a Manutenção
  2. Software modularizado
  3. + 7500 extensões
  4. Qualquer um pode desenvolver um template para joomla
  5. Mais de 64 idiomas de instalação
  6. Prêmio de Melhor CMS dos anos 2006, 2007 e 2011 (OPEN SOURCE CMS AWARD)

O que significa o termo Joomla?

O nome Joomla vêm do equivalente fonético da palavra Swahili “Jumla”, que significa “todos juntos” ou “como um todo”.

  1. Licença GNU
  2. Segurança: Joomla Strike Team
  3. Suporte - mais de 200 mil usuários em todo mundo

Joomla Framework

  1. Framework de desenvolvimento em PHP
  2. Encontra-se em versão 13.1
  3. Desenvolvido de forma independente do Joomla CMS
  4. O Joomla CMS é construído utilizando o framework
  5. Posso fazer um outro CMS utilizando o Joomla Framework
